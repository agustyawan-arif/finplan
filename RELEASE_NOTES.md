# Release Notes — Pockit

## Initial Release: Pockit v0.1.0 — Private Beta

Welcome to **Pockit**! Pockit is a premium, mobile-first money manager designed to track budgets, accounts, investments, and net worth under a clean, physical ledger style.

### Features
- 🔒 **Email Login and Protected App:** Secure user registration, authentication, and session persistence backed by Supabase Auth.
- 💾 **Persistent Personal Finance Data:** Fully database-backed personal finance data, strictly secured and isolated using PostgreSQL Row-Level Security (RLS) policies.
- 💳 **Account and Pocket Tracking:** Real-time balance calculations, multi-currency wallets, savings pocket separations, and active account management.
- 💸 **Transaction Tracking:** Clean forms for recording daily Income, Expense, and internal Transfer transactions, with automatic default conversion fallback ratios.
- 📊 **Budget Management:** Custom monthly spending limits for logic categories (Needs, Wants, Charity) and allocation tracking for savings.
- 📈 **Investment and Asset Tracking:** principal costs, average buy pricing, and stock/mutual fund holding revaluations.
- 📉 **Reports and Charts:** High-utility comparative charts for Budget vs. Actual categories and Net Cashflow metrics.
- 📱 **Mobile-First UI:** Responsive iOS simulator shell optimized with Home Indicator safe-areas, smooth micro-animations, and soft ambient elevation tokens.
